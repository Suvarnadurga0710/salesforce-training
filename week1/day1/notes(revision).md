# What problem does Salesforce solve?
Salesforce helps companies manage customer relationships, sales, support, and business operations in one platform. It solves problems related to customer tracking, communication, workflow management, and data organization.
# What is CRM?
CRM stands for Customer Relationship Management. It is a system used to manage customer information, interactions, sales processes, and support activities.
# What is an Object in Salesforce? Give one example.
An Object in Salesforce is a database table used to store information.
Example:
Account Object → stores company/customer details.
# Difference between Salesforce Admin and Developer
Salesforce Admin	Salesforce Developer
Manages Salesforce setup and users	Builds custom applications
Uses configuration tools	Uses programming and coding
Creates reports and workflows	Writes Apex and Lightning code
Mostly no-code customization	Advanced customization and integrations
