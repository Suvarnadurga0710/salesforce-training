# Day 4 - Flow Builder Basics

## What is Flow Builder?

Flow Builder is a Salesforce tool used to automate business processes without writing code. It helps users create workflows using a visual interface.

## Types of Flows

### Screen Flow

A Screen Flow interacts with users through screens and collects information.

Example:

* Student Registration Form

### Record Triggered Flow

A Record Triggered Flow runs automatically when a record is created or updated.

Example:

* Send a confirmation email after student registration.

## Automation Ideas for College Management System

1. Send email after student registration.
2. Generate student ID automatically.
3. Send fee payment reminders.
4. Notify faculty when a course is full.
5. Update available seats automatically.

## Manual vs Automated Process

### Process: Student Registration

#### Manual Process

* Student fills the form.
* Staff checks the details.
* Staff enters data manually.
* Staff sends confirmation email.

#### Problems

* Takes more time.
* Chances of mistakes.
* Delayed communication.

#### Automated Process

* Student submits the form.
* Salesforce creates the record automatically.
* Confirmation email is sent automatically.

#### Benefits

* Saves time.
* Reduces errors.
* Faster process.

## Reflection

Automation helps companies save time and reduce repetitive work. It improves productivity and makes business processes more efficient.

## Answers to Reflective Questions

### Why do companies automate workflows?

To save time and improve efficiency.

### What problems happen with manual processes?

They are slow and can lead to mistakes.

### Difference between Screen Flow and Record Triggered Flow?

Screen Flow requires user input, while Record Triggered Flow runs automatically.

### Why is no-code automation powerful?

People can build automation without programming knowledge.

### When should automation be avoided?

When human decision-making is required.

### How does automation improve consistency and productivity?

It performs tasks the same way every time and reduces manual effort.
